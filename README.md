# 📱 Assignment 3 - Mobile Application Development
Name: Alina Mubarik  
Course: Mobile App Development  

 Project Overview
This repository contains multiple mobile applications developed using **React Native (Expo)**.  
Each application demonstrates frontend UI, navigation, and backend integration using **SQLite** and **AsyncStorage**.
Applications Included

 1. E-Commerce App
Features:
- Product listing from SQLite database
- Product details screen
- Add to Cart using AsyncStorage
- Loading and error handling

2. Social Media App
Features:
- Feed screen displaying posts
- Post details screen
- Profile screen with edit functionality
- Data stored and updated using SQLite

3. Ticket Booking App
Features:
- Movies list from SQLite
- Search functionality
- Seat selection system
- Save selected seats using AsyncStorage

 4. Expense Tracker App
Features:
- Dashboard showing transactions
- Income, Expense, and Balance calculation
- Reports screen with chart visualization

5. Food Delivery App
Features:
- Restaurant listing
- Filter by cuisine
- Order history screen
- Navigation between screens

Technologies Used
- React Native (Expo)
- SQLite (expo-sqlite)
- AsyncStorage
- React Navigation
- React Native Chart Kit

 How to Run the Apps:

1. Open any app folder:
   cd app-name
      
2. Install dependencies:
   npm install
   
3. Start the app:
   npx expo start

4. Scan QR code using Expo Go.

 Key Concepts Implemented
- Component-based architecture
- Navigation using React Navigation
- Local database using SQLite
- Local storage using AsyncStorage
- State management using React Hooks
